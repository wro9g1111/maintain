## [You Don't Need](https://github.com/you-dont-need/You-Dont-Need)
특정 도구/라이브러리/패턴들의 사용을 대체할 수 있는 방법(코드 예제)을 제공한다.

ex) Underscore/Lodash의 `_.dropRight()`은 다음의 네이티브 코드로 대체할 수 있다.
```js
_.dropRight([1, 2, 3]);
// => [1, 2]

_.dropRight([1, 2, 3], 2);
// => [1]

// Native
[1, 2, 3].slice(0, -1);
// => [1, 2]

[1, 2, 3].slice(0, -2);
// => [1]
```
